## 生成插件文档
```plaintext
请通读整个插件的源码，全面理解其功能、配置项、API、命令、键位映射等。然后，生成一份详尽且具体的文档，内容包括：

   1. 插件的主要功能和用途简介；
   2. 如何通过 lazy.nvim 进行插件的安装和调用（包括最简配置和高级配置示例）；
   3. 插件的所有可用配置项及其详细说明；
   4. 常用命令、键位映射及其作用；
   5. 典型使用场景和示例代码；
   6. 任何注意事项或常见问题。
```
